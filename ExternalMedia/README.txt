- ExternalMedia -


Store any external media not yet used directly in the UnityProject
DO NOT LET THIS GET BLOATED, when an asset is moved into the unity project remove it from here
It is additionally recommended to keep a local copy of any asset you make in case of any emergency rollbacks

* Code *
	- Any sort of script or source code
	
* Images *
	- All types of 2d assets
	
* Models *
	- 3d assets potentially including their textures if required
	
* Sounds *
	- Any sound or music asset
